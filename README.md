# Employee Attrition Prediction

## 📌 Project Overview

Employee Attrition Prediction using Weka. Includes data cleaning, visualization, model building, and analysis of overfitting and underfitting using real HR dataset.

## 🛠️ Tools Used

* Weka
* Google Sheets
* Flourish
* Napkin AI

## 📊 Data Preprocessing

* Cleaned missing values
* Converted Attrition (Yes/No → 1/0)

## 🤖 Model Building

* Algorithm: Random Forest
* Tool: Weka
* Evaluation: 10-fold cross validation

## 📈 Results

* Achieved good accuracy in predicting employee attrition
* Overtime and low salary are major factors

## 📊 Visualizations

* Charts created using Flourish
* Analysis done in Google Sheets

## 🧠 Insights

* Employees working overtime are more likely to leave
* Younger employees show higher attrition

This project helps in understanding employee behavior and provides insights to reduce attrition.
